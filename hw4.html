Types of the List

Lists Usage
Unordered List Each item is shown, by default, with a bullet. An unordered list is used to present a list of items that not necessarily need to be shown in any specific order. To start this list you use the tag ‹ul›
Ordered List Each item is shown, by default, with a number. An ordered list is generally used to present a list of items that need to be shown in a specific order. To start this list you use the tag ‹ol›
Definition List The items come in pairs. A definition list is used to define the term and the definition (description).To start this list you use the tag ‹dl›
